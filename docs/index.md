# Hyperswitch Documentation

Hyperswitch is a high-performance, open-source payment orchestration platform that lets you connect to multiple payment processors through a single API.

## 🛠️ Built for Developers

Hyperswitch provides a unified infrastructure for global payments:

- **Unified API** - One integration for 50+ processors.
- **Smart Routing** - Optimize for success rates and processing fees.
- **Real-time Analytics** - Deep insights into your payment health.
- **100% Open Source** - Self-hostable, extensible, and transparent.
