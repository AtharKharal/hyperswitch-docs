# Events - Delivery Attempt List

**Endpoint:** `GET /events/{merchant_id}/{event_id}/attempts`

List all delivery attempts for the specified Event.

**Operation ID:** `List all delivery attempts for an Event`

## Playground

<a href="https://sandbox.hyperswitch.io/playground?endpoint=/events/{merchant_id}/{event_id}/attempts&method=GET" class="md-button-playground" target="_blank">Try in Playground →</a>

## Code Examples

=== "cURL"

```bash
curl -X GET "https://sandbox.hyperswitch.io/events/{{merchant_id}}/{{event_id}}/attempts" \
  -H "Content-Type: application/json" \
  -H "api-key: YOUR_API_KEY"
```

=== "Python"

```python
import requests

url = "https://sandbox.hyperswitch.io/events/{merchant_id}/{event_id}/attempts"
headers = {"Content-Type": "application/json", "api-key": "YOUR_API_KEY"}
response = requests.get(url, headers=headers)
print(response.json())
```

=== "JavaScript"

```javascript
const axios = require("axios");
const url = "https://sandbox.hyperswitch.io/events/{merchant_id}/{event_id}/attempts";
const options = {
  method: "GET",
  url: url,
  headers: {"Content-Type": "application/json", "api-key": "YOUR_API_KEY"}
};
axios.request(options).then(r => console.log(r.data)).catch(e => console.error(e));
```
